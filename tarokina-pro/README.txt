== Description ==
## Intuitive and easy to use. Provides accurate tarot readings on WordPress.
Design and display tarot readings from 1 to 10 cards.
Basic, Reversed, and Expert mode. If the cards have different meanings in different positions, you can define each card for each position using the expert mode.
